# 🎰 Slot Machine Game (Python)

A simple terminal-based Slot Machine game built with Python.

## 📌 Features

* Deposit money to start playing
* Choose how many lines to bet on (1–3)
* Set bet amount for each line
* Random slot machine spin (3x3 grid)
* Winning line detection
* Payout system based on symbols
* Play multiple rounds without restarting

## 🛠 Built With

* Python 3
* Random module

## 🎮 How to Play

1. Run the program
2. Deposit money
3. Press Enter to play
4. Choose number of lines
5. Enter bet amount per line
6. Spin the machine
7. Win money if symbols match across a line
8. Press `q` anytime to quit

## 💰 Symbol Values

| Symbol | Value |
| ------ | ----- |
| A      | 5x    |
| B      | 4x    |
| C      | 3x    |
| D      | 2x    |
